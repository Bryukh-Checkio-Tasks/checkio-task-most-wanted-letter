You are given a text, which contains different english letters and punctuation symbols. You should to find the most frequent letter in the text. The letter returned must be in lower case.
While checking for the most wanted letter, case does not matter so for the purpose of your search, "A" == "a". Do not count punctuation symbols and whitespaces, only letters. If you have two or more letters with the same frequency, then return the letter which comes first in the alphabet. (Ex: we have "b" and "f" as the most frequent, then return "b")

Input: A text for analysis. A string (Unicode).

Output: The most frequent letter. A string.

Example: 

checkio("Hello World!") == "l"
checkio("How do you do?") == "o"
checkio("One") == "e"

How it is used: For most of decryption tasks where you need to know the frequency of occurrence for various letters in the text. For example, we can easily crack simple addition or substitution cipher if we know the frequency of appearing the letters. This information may be interesting to know for language experts.
